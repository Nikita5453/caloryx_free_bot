# Caloryx Free Bot

Бесплатный Telegram-бот для подсчета калорий и ведения дневника питания.

## Что умеет

- Создает профиль пользователя: пол, возраст, рост, вес, активность и цель.
- Считает дневную норму калорий по формуле Mifflin-St Jeor.
- Ведет дневник питания в SQLite.
- Принимает быстрые записи вида `гречка 200`, `курица 150`, `яблоко`.
- Показывает остаток калорий на день и историю.
- Не содержит платного доступа, подписок, платежей и лимитов.

## Запуск

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

3. Создайте `.env` рядом с `bot.py`:

```env
BOT_TOKEN=ваш_токен_бота
```

4. Запустите:

```powershell
python bot.py
```

## Кнопка «Сканировать» и Mini App

В боте добавлена синяя кнопка `📸 Сканировать`. Она открывает Telegram Mini App из папки `webapp`.

Telegram открывает Mini App только по публичной `https://` ссылке. Поэтому файл `webapp/index.html` нужно загрузить на хостинг, например Netlify, Vercel, GitHub Pages или свой сервер.

После публикации добавьте ссылку в `.env`:

```env
BOT_TOKEN=ваш_токен_бота
WEB_APP_URL=https://ваш-сайт/webapp/index.html
```

После изменения `.env` перезапустите бота:

```powershell
python bot.py
```

## Добавление еды через ИИ

Mini App отправляет описание блюда на backend бота:

```text
http://127.0.0.1:8787/api/analyze-food
```

Для работы ИИ добавьте в `.env`:

```env
OPENAI_API_KEY=ваш_openai_api_key
OPENAI_MODEL=gpt-4.1-mini
WEBAPP_API_HOST=127.0.0.1
WEBAPP_API_PORT=8787
```

После этого перезапустите бота. В Mini App нажмите `+ Добавить`, опишите блюдо, например `гречка около 200гр`, и нажмите `Добавить`.

## Команды

- `/start` - главное меню
- `/profile` - настроить профиль
- `/add` - добавить прием пищи
- `/today` - дневная сводка
- `/history` - последние записи
- `/reset` - очистить профиль и дневник

## Примеры записей еды

```text
овсянка 250
курица 180
банан
рис 200
творог 150
```

Если бот не знает продукт, он попросит указать калорийность вручную.
